how i work

1

problem analisys

saya akan menganalisa permasalahan terlebih dahulu. misalnya pada website toko online, bagaimana cara user melakukan transaksi, apa yang mau ditampilkan ketika pertama kali website dibuka, apa yang terjadi setelah user melakukan transaksi, dll.

2

designing solutions

kemudian saya akan merancang solusi dari permasalahan tersebut secara garis besar beserta membuat algoritmanya. misalnya pada website toko online, ketika user membuka website maka tampilkan halaman home yang berisi katalog produk, lalu ketika user memilih produk maka akan tampil halaman detail, lalu tersedia tombol order, lalu user diarahkan ke halaman pembayaran ketika mengklik order, dan harus melakukan pembayaran dan upload bukti agar pengiriman bisa diproses.

3

create a module

setelah memikirkan algoritmanya seara garis besar, saya akan memikirkan hal-hal yang lebih detail dan membaginya ke dalam modul terpisah. misalnya modul user yang berisi fitur untuk user, modul admin yang berisi fitur untuk admin, modul produk yang berisi CRUD produk, dll.

4

excecution

setelah itu saya akan mulai mengeksekusi pembuatan program. dimulai dari membuat tampilan, hingga bermain dengan database dan logika.

5

trials

setelah program dibuat, saya akan menguji cobanya untuk mencari bug dan memastikan program berjalan sesuai yang diharapkan sebelum akhirnya dihosting.
